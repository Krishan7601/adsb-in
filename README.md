### Download
Clone repository:  
`git clone https://github.com/Krishan7601/adsb-in.git`  
Pull latest version of all submodules:  
`git pull --recurse-submodules`  
### Installation
### Node App Development
Run app in development mode:  
`DEBUG=display-data:* npm run devstart`  
Execute unit tests:  
`npm test`
